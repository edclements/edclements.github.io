--- title: Command Reference layout: default ---

Terminal

CTRL-T - Fuzzy search for files

DWM

MOD-SHIFT-ENTER - Open terminal window

MOD-SHIFT-C - Close window

MOD-P - Launch dmenu

MOD-SHIFT-Q - Quit DWM

MOD-L - Lock screen

Vim

:set list - Show whitespace

:syntax sync fromstart - Fix syntax highlighting

Audio

Start JACK

jack_control start

Stop JACK

jack_control stop
